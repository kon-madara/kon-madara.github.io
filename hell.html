納魂粥（のうこんがゆ）

魂を鎮め、身を癒す──地獄の台所より届けし、黄泉の朝餉。

材料（1人前）

  • オートミール … 30g（米化タイプ推奨）
  • 水 … 120ml（粥状になるまで調整可）
  • 納豆 … 1パック
  • 卵 … 1個
  • 醤油 … 小さじ1（好みで調整）
  • 白だし or 和風だしの素 … 小さじ1/2（任意）
  • 青ねぎ・ごま・刻み海苔 … お好みで
  • 狐火のごとき祈念 … 適量

作り方

  1. 小鍋に水を入れ、オートミールを加える。中火で火を入れ、時折かき混ぜる。
  2. 粥状になったら火を弱め、納豆をそのまま投入し、全体になじませる。
  3. 卵を割り入れ、黄身がほどよく固まるまで煮る。蓋をして1分ほど蒸らしてもよい。
  4. 醤油と白だしで味を整える。必要であれば、塩をひとつまみ。
  5. 器に盛り、青ねぎ・ごま・海苔をのせる。狐火が揺らぐような湯気が立ちのぼれば完成。

効能

  • 内臓にやさしく、心にあたたかい
  • 塩分・脂質控えめ、満腹感あり
  • 一杯で「現世と地獄の狭間」へ誘う粥

静穏葬送麺（せいおんそうそうめん）

地獄の底にも静けさはある──豆もやし一袋で魂を穏やかに。

材料

  • 豆もやし … 1袋
  • 鶏がらスープの素 … 適量
  • 塩 … 適量
  • 醤油 … 適量
  • ニンニクチューブ … 少々
  • （お好みで）ねぎ・卵など

作り方

  1. 豆もやしをレンチンする（目安は600wで3分）。
  2. その間にスープを用意（全て目分量、味見は必ず）。
  3. スープに豆もやしを投入。
  4. お好みでねぎ・卵などをトッピング。
  5. ※スープは好みで自由にアレンジ可能。
